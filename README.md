# Opportune

Opportune is a React-based web app for finding jobs and internships.  
It allows users to search for roles, apply basic filters, sort results, and save listings they find useful.

---

## Features

- Search jobs and internships  
- Filter based on preferences  
- Sort results (latest, relevance, etc.)  
- Save jobs for later  
- Fetch data from an external API  

---

## Tech Stack

- React  
- JavaScript  
- Fetch API  
- Tailwind CSS  
- React Router
